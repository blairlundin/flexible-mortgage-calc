# Feature Suggestions

## High Value
1. **Comparison mode** — Allow users to compare two scenarios side-by-side (e.g., different rates, terms, or extra repayment amounts) to see the impact
2. **Offset account simulation** — Model an offset account balance that reduces the interest-bearing principal
3. **Interest saved from extra repayments** — Show how much interest and time the user saves compared to the base repayment (this is the key motivator for extra repayments)
4. **Variable rate periods** — Allow defining multiple rate periods (e.g., 2 years fixed at 5.5%, then variable at 6.2%) since most real mortgages have rate changes

## Medium Value
5. **Print / PDF export** — Allow exporting the amortization schedule and chart
6. **Lump sum payments** — Support one-time extra payments at specific points in the loan (e.g., "pay $10k extra in year 3")
7. **Stamp duty & fees calculator** — Add upfront cost estimation (stamp duty, LMI, conveyancing) as a companion tool
8. **Dark mode** — Add theme toggle

## Quality of Life
9. **Input sliders** — Add range sliders alongside number inputs for principal, term, and rate for more intuitive adjustment
10. **URL sharing** — Encode form state in URL params so users can share/bookmark specific scenarios
11. **Mobile responsiveness** — The current `Col sm="2"` layout likely breaks on small screens; improve the responsive grid
12. **Accessibility** — Add ARIA labels, keyboard navigation for button groups, and proper form validation feedback
