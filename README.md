# Flexible Mortgage Calculator

A mortgage calculator built with React, TypeScript, and Vite. Supports adjustable-rate mortgages and visualizes payment schedules with interactive charts.

## Features

- Fixed and adjustable-rate mortgage calculations
- Amortization schedule with principal/interest breakdown
- Interactive charts powered by Chart.js
- Responsive UI using Bootstrap and Reactstrap

## Tech Stack

- **React 19** with TypeScript
- **Vite** for bundling and dev server
- **Chart.js** / **react-chartjs-2** for charts
- **Bootstrap 5** / **Reactstrap** for UI components
- **Vitest** for unit tests

## Available Scripts

### `npm start`

Starts the development server at `http://localhost:5173`.

### `npm run build`

Builds the app for production to the `dist` folder.

### `npm run preview`

Serves the production build locally for preview.

### `npm test`

Runs the test suite with Vitest.

### `npm run lint`

Lints the source files with ESLint.
